# AI_RULES (always follow)

1) Session start:
- Read Docs/PLAN.md and Docs/TASKS.md
- Summarize today's goal in 5 bullets

2) Before coding:
- Add a task item to Docs/TASKS.md with acceptance criteria

3) After any change:
- Append to Docs/WORKLOG.md:
  - date/time
  - what changed
  - files touched
  - next step
  - risks

4) Decisions:
- Any non-trivial choice -> log in Docs/DECISIONS.md
